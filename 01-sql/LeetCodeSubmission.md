# LeetCode SQL Practice
The LeetCode SQL questions are meant to help you gain confidence and fluency with SQL. Reading about SQL or doing a few lab exercises won't build the permanent learning you need — consistent practice will. The goal is to work through all 77 free SQL problems until you can consistently solve "Easy" and some "Medium" questions without hints.

**Note:** Practicing SQL on LeetCode will help you think about how to join data on any platform, including NoSQL. Clear thinking about joins is a transferable skill that benefits you even in designing stateless applications.

---

## Problem List
Use the following link to find database problems to practice on and solve:

### Study Tips
- Try resetting problems and solving them without looking up syntax or solutions
- Practice with a timer to simulate real interview conditions
- You can ask AI for help and look at others' solutions while learning
- Focus on eventually solving problems independently
